# Linux Access Control Lab – Pharmacy System

This lab simulates a small pharmacy system to practice Linux system administration tasks including user creation, group management and file permissions (oracle VirtualBox, Ubuntu)

## Objectives

- Create users and groups
- Assign file permissions
- Configure access controls
- Validate user access

  ## Role-Based Access Control
  
ROLE            DIRECTORY        PERMISSIONS
Pharmacists     prescriptions    read, write, execute
Assistants	    inventory	       read, write, execute
Billing        	invoices	       read, write, execute
